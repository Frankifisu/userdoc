
======================
MOPAC Manual |release|
======================

MOPAC [:ref:`1<reference 1>`] is a general-purpose semiempirical quantum chemistry engine for the study of molecular and periodic structures. This manual documents the MOPAC engine in the |release| release of the Amsterdam Modeling Suite. The MOPAC engine shares the core routines with the `standalone MOPAC program <http://openmopac.net/index.html>`__ and can be used via the `AMS driver <../AMS/index.html>`__.
